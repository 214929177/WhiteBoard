# WhiteBoard
1、涂鸦功能，包括画笔粗细选择、颜色选择、橡皮擦等；  
2、文字输入，包括颜色选择、增、删、改等；  
3、白板功能，包括撤销、重做、新增、锁屏、切换、保存为图片、保存到本地等功能
